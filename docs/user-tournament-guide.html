How Tournaments Work

The full flow from creation to prize payout

1
Organizer creates a tournament Sets rules, entry fee, schedule, prize pool, and bracket format.
2
Players register and pay entry fee Any eligible user can sign up before the registration deadline.
3
Matches run per bracket or round-robin System automatically schedules and tracks all matches.
🏆

ℹ️ Think of a tournament as a structured competition — same rules, same schedule for everyone. Fair and transparent.

Public vs Private Tournament

Pick the right visibility for your event

🌐 Public Tournament

  • Visible to all users in the app.
  • Anyone can register.
  • Best for large participation.
  • Appears on the main tournament listing.

🔒 Private Tournament

  • Visible only to invited users.
  • Requires an Invite Code.
  • Best for small groups or clubs.
  • Shown on the Private Tournament page.

Tournament Creation Form

Every field explained — what to fill and why

Tournament Name
The title shown to players. Helps users identify the event.
Game
Select the game (Ludo). Ensures correct rules and bracket format.
Entry Fee
Amount a player pays to join. This builds the prize pool. Set 0 for a free tournament.
Prize Pool / Rewards
What winners receive. Motivates players to participate.
Max Players
Total players allowed. Keeps the bracket balanced.
Players Per Match
How many players compete per match (2 or 4).
Format
Knockout, Round Robin, Double Elimination, or Group + Knockout.
Start Time
When the tournament officially begins. All players must be ready by this time.
Registration Deadline
Last date/time to register. Locks participants before the start.
Tournament Type
Public or Private. Controls who can see and join.
Invite Code Private only
Required for private tournaments. Share this code with invited players.
Status
Draft / Published / Closed — controls visibility and registration.
Description / Rules
Clear rules help avoid confusion and disputes.
Platform Fee %
Percentage deducted as platform service charge from the prize pool.
Prize % (1–5)
How the prize pool splits among the top 5 finishers. Values must total 100%.
💡 If Entry Fee is set to 0, the tournament is completely free for all players.
