Bonus + Commission Auto Calculation

Laravel command added to process legacy pending purchases, deposit bonuses, referral bonuses, and extra bonus credits.

  • Command: php artisan legacy:process-bonus-commission
  • Scheduled: runs every minute via routes/console.php
  • Supports nowpayments + payformee checks.

Env keys needed:

  • PAYMENTAPI_KEY (nowpayments)
  • PAYFORMEE_USER_TOKEN (payformee)
  • LEGACY_INCOME_DEPOSIT_BONUS=true to enable deposit bonus rules

Writes to legacy tables: tbl_purchase, tbl_users, tbl_statement, tbl_direct_admin_profit_statement, tbl_purcharse_ref, tbl_referral_bonus_log, tbl_extra_wallet_log.
